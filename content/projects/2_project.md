Designed a real-time robot control framework using EtherCAT as the core hardware interface, supporting multi-protocol conversion and high-precision, synchronized industrial control. Developed hardware selection, slave firmware, and Linux-based master stack with ROS integration.

## Highlights

- Led hardware selection, slave firmware development, and protocol conversion (EtherCAT↔CAN/UART/RS485).
- Conducted extreme condition stress tests and designed safety/maintenance strategies to industrial standards.
- Built Linux x86 EtherCAT master stack with multi-thread/process scheduling and ROS interface.
- First inventor, patent published: "Real-Time Robot Control System Based on EtherCAT Master Interface".
